弹出层

一、 功能简介
在页面上弹出浮动层，支持拖动、遮罩、加载动画等，可以弹出多级。 用于取代浏览器本身的弹出网页和对话框，使整体样式统一。
  • 复杂页面/共用页面需要使用iframe内嵌
  • 静态内容较短时使用页面内嵌，否则使用JSP内嵌
  • 单独页面使用，且需要Ajax刷新的，使用JSP内嵌
二、实例说明
1. 页面内嵌
将页面中的某容器(通常是div)作为内容显示在弹出层中。
实例演示：

输入100才能关闭


 
   
1.1 网页内容
将内容写在div中，并设定class="none"来隐藏它。
建议放在页面最后，初始化弹出层时将被移动到弹出层中。
HTML：
												<div class="none" id="myPopContent">
													这是弹出层的内容
												</div>
1.2 初始化弹出层
弹出层必须先进行初始化，之后才可以进行弹出、关闭等操作。
初始化的过程建议写在onload事件中，弹出层对象需要声明为全局变量。
以下未加*的项目可以不设定，默认值为false或null。
Javascript：
												myPopbox = new PopupBox({
													// *唯一标志，相同页面中不可重复
													key: 'my1',
													
													// *标题内容，可用元素或字符串
													title: '测试用标题',
													
													// *图标的CSS
													icon: 'img_opt opt_Help',
													
													// *内容元素
													content: $('myPopContent'),
													
													// *显示位置，相当与z-index
													position: 3,
													
													// 是否允许拖动
													drag: true,
													
													// 关闭前的回调，通常用于校验等
													beforeclose: function() {
														// 如果返回false不会关闭弹出层
														return checkSomething();
													},
													
													// 关闭后的回调，通常用于刷新页面等
													afterclose: function() {
														loadData();
													}
												});
1.3 弹出 & 关闭
Javascript：
												// 弹出在默认位置(居中)
												myPopbox.popup();
												
												// 弹出在指定位置(top:200,left:100)
												myPopbox.popup(200, 100);
												
												// 弹出在指定位置，不带遮罩
												myPopbox.popup(200, 100, true);
												
												// 弹出在默认位置，不带遮罩
												myPopbox.popup(null, null, true);
												
												// 带回调的关闭
												myPopbox.close();
												
												// 不带回调的关闭(强制关闭)
												myPopbox.close(0);

3. iframe内嵌
弹出层中显示另一个页面。
实例演示：

3.1 父页面内容
将一个iframe写在div中，可以先不写src属性，需要加载时再赋值。
使用$(yourFrame).contentWindow来获取子页面的命名空间。
HTML：
												<div class="none" id="myPopContent" >
													<iframe id="myInnerPage03"></iframe>
												</div>
3.2 子页面内容
子页面中需要正常引用所需的css和js，使用parent来获取父页面命名空间。
如果使用了loader:true，子页面加载完成后，需要通知父页面。
HTML：
												<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
												<html>
													<head>
														<meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
														<title>子页面</title>
														<!-- 共通css -->
														<link rel="stylesheet" type="text/css" href="../css/gray.css">
														<!-- 共通js -->
														<script type="text/javascript" src="../js/common/prototype.js"></script>
														<script type="text/javascript">
															// 在加载完成时调用父页面接口
															Event.observe(window, 'load', function() {
																parent.myInnerPageLoaded();
															});
															function close1() {
																// 调用父页面接口
																parent.myInnerPageClose();
															}
															
															function close2() {
																// 调用父页面接口
																parent.myInnerPageCloseForce();
															}
														</script>
													</head>
													<body>
														<div class="span-3 last">
															<p>输入99才能关闭</p>
															<input type="text" id="input03"/>
															<div class="text_right span-3 last">
																<input type="button" value="确定" class="btn span-2" onclick="close1();"/>&nbsp;
																<input type="button" value="取消" class="btn span-2" onclick="close2();"/>
															</div>
														</div>
													</body>
												</html>
3.3 初始化弹出层
初始化方式同页面内嵌，但一般设置loader:true来表示加载过程。
同时还要提供接口，在子页面加载完成以及关闭时都需要调用。
这些接口应在子页面中规定好，所有弹出此页面的父页面都需要实现接口所需的方法。
Javascript：
												var myPopbox03;
												Event.observe(window, 'load', function(){
													myPopbox03 = new PopupBox({
														// *唯一标志，相同页面中不可重复
														key: 'my03',
														
														// *标题内容，可用元素或字符串
														title: '测试用标题',
														
														// *图标的CSS
														icon: 'img_opt opt_Help',
														
														// *内容元素
														content: $('myPopContent03'),
														
														// *显示位置，相当与z-index
														position: 3,
														
														// 是否允许拖动
														drag: true,
														
														// 是否需要加载动画
														loader: true,
														
														// 关闭前的回调，用于校验等
														beforeclose: function() {
															// 如果返回false不会关闭弹出层
															return beforeClose03();
														},
														
														// 关闭后的回调，用于刷新页面等
														afterclose: function() {
															afterClose03();
														}
													});
												});
												
												// 接口：加载完成
												function myInnerPageLoaded() {
													myPopbox03.loaded();
												}
												
												// 接口：带回调关闭
												function myInnerPageClose() {
													myPopbox03.close();
												}
												
												// 接口：无回调关闭
												function myInnerPageCloseForce() {
													myPopbox03.close(0);
												}

4. 注意事项
  • 不要使用相同的参数重复生成弹出层
  • 参数key相当于弹出层的id,每个弹出层都应是不同的
  • 如果使用了Loader，一定记得在ajax的oncomplete或嵌入页的onload中调用弹出层的loaded()方法
  • 弹出层右上角的关闭是不带回调的，并且可以设置noclose:true来隐藏它
  • 你可以在确定按钮的事件中校验，而不是必须用回调
  • 子页面调用父页面：parent.某方法()，例：parent.$('myDiv')
  • 父页面调用子页面：$('xxFrame').contentWindow.某方法()，例：$('xxFrame').contentWindow.testFunc(123)
三、 方法接口
1. 构造参数
new PopupBox(paramObject)
  • key:string 唯一标志，任意字符串
  • title:string|element 弹出层标题
  • content:element 弹出层内容
  • position:int z轴位置
  • icon:string 图标class
  • drag:bool 是否可拖动
  • loader:bool 是否有加载动画
  • noclose:bool 是否去掉关闭按钮
  • beforeclose:function 关闭前回调，必须有返回值(true:关闭,false:不关闭)
  • afterclose:function 关闭后回调
2. 方法
2.1 popup([top, left][, nocover = false]) 弹出窗口
  • top:int 弹出层上边距
  • left:int 弹出层左边距
  • nocover:bool 是否去掉遮罩
2.2 close([callbackMode = 3]) 关闭窗口
  • callbackMode:int 回调执行选项(0:无，1:仅关闭前，2:仅关闭后，3:全部回调)
2.3 loaded() 使用loader时，页面加载完成后调用
3. 属性
3.1 beforeclose:function 关闭前回调方法
3.2 afterclose:function 关闭后回调方法
