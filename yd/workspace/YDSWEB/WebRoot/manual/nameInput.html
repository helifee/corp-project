员工姓名自动匹配

一、 功能简介
提供对员工的查找，智能候补输入功能。
可以通过输入员工ID、 姓名汉字（前方一致）、 姓名拼音首字母（前方一致），
对员工姓名进行匹配，得到符合条件的候选员工下拉列表以供选择。

1.网页内容
引入JsInputFilter.js，JsNameFilter.js。JSP中书写姓名ID和汉字的入力框，设定好name和id，可以自由布局，。
html：
												<body onload="init()">
												员工ID： <input type="text" id="idInput" name="idInput"/>
												员工姓名： <input type="text" id="nameInput" name="nameInput" />
2.JS内容
在页面加载时（一般是onload函数），执行共通JsNameFilter函数，为目标控件输入框绑定事件。
"../"为相对路径，页面在WebRoot下第几层就加几个"../"
Javascript：
												function init() {
												    new JsNameFilter("idInput", "nameInput", "../");
												}
三、 方法接口
初始化方法
JsNameFilter(idInputId, nameInputId, pathStr, notCheckInput) 通过输入内容过滤出相应的员工
  • idInputId:string 员工ID输入框ID
  • nameInputId:string 员工姓名输入框ID
  • pathStr:string action相对路径
  • notCheckInput:bool 是否验证ID输入
