消息对话框

一、 功能简介
用弹出层实现MsgBox.message和confirm的功能，使其样式与网站整体设计统一。
二、示例说明
1. 弹出提示
1.1 仅内容
Javascript：
												MsgBox.message('消息内容');
1.2 内容及标题
Javascript：
												MsgBox.message('消息内容', '消息标题');
1.3 内容、标题 & 关闭时回调
Javascript：
												MsgBox.message('消息内容', '消息标题', function(){
													// 确定时回调
													MsgBox.message('执行了关闭后事件！');
												});
2. 弹出错误
2.1 仅内容
Javascript：
												MsgBox.error('XXX错误');
2.2 内容及标题
Javascript：
												MsgBox.error('XXX错误', '错误标题');
2.3 内容、标题 & 关闭时回调
Javascript：
												MsgBox.error('XXX错误', '错误标题', function(){
													// 确定时回调
													MsgBox.message('执行了关闭后事件！');
												});
3. 弹出确认对话框
标准调用
Javascript：
												MsgBox.confirm('是否要去吃饭？', '确认对话框', function(){
													// 确定时回调
													MsgBox.message('你选择了Yes！');
												}, function(){
													// 取消时回调
													MsgBox.message('你选择了No！');
												}, 'Yes', 'No');
三、 方法接口
1. 方法
1.1 MsgBox.message(content[, title[, onclose]]) 信息对话框
  • content:string 内容
  • title:string 标题
  • onclose:function 关闭后回调
1.2 MsgBox.error(content[, title[, onclose]]) 错误对话框
  • content:string 内容
  • title:string 标题
  • onclose:function 关闭后回调
1.3 MsgBox.confirm(content[, title[, onOk, onCancel[, btnOk, btnCancel]]]) 确认对话框
  • content:string 内容
  • title:string 标题
  • onOk:function 确定按钮回调
  • onCancel:function 取消按钮回调
  • btnOk:string 关闭后回调
  • btnCancel:string 关闭后回调
