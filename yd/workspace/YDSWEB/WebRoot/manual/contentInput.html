输入自动匹配组件

一、 功能简介
实现类似于百度的候补输入效果，通过用户输入，利用设置好的候选内容，进行自动匹配并显示候选下拉框。
候选内容可以通过访问数据库获得。
  • 需要引用 js/common/JsInputFilter.js js/common/JsContentFilter.js
二、示例说明
示例
实例演示：

预先设置的候选内容为 {"大连","大连市","大连甘井子","辽宁","辽宁大连","dalian","da"}

1. HTML 代码
html：
												<body onload="init()">
												<input id="contentInput" type="text"/>
2. Js 代码
"contentInputFilter.action" 是得到候选内容请求的ActionUrl（以json的形式返回）
"tempArray"为action中存储候选内容List的变量名
Javascript：
												function init() {
													new JsContentFilter("contentInput", "contentInputFilter.action", "tempArray");
												}
三、 方法接口
初始化方法
JsContentFilter(contentInputId, actionUrl, arrayName) 设定输入自动匹配
  • contentInputId:string 输入控件ID
  • actionUrl:string 请求Action的url
  • arrayName:string 储存过滤内容的list名
